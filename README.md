# get_SberZvuk
SberZvuk download PS script

Скрипт можно запускать из любого места системы.
При первом запуске он запросит данные для формирования конфига.

Пример файла config.json
```
{
    "email":  "",
    "password":  "",
    "token":  "you token",
    "format":  3,
    "lyrics":  true,
    "outPath":  "d:\\music\\Down\\",
    "maxCover":  "500x500",
    "formatFallback":  true
}

```
